# DROP DATABASE
Deletes a database.

## Syntax
{: .no_toc}

Deletes the database and all of its tables and attached engines.

```DROP DATABASE [IF EXISTS] <database_name>```

## Parameters
{: .no_toc}

| Parameter         | Description                            |
| :----------------- | :-------------------------------------- |
| `<database_name>` | The name of the database to be deleted |
